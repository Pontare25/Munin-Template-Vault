# Log

Changelog for the Atlas knowledge bundle ([OKF §7](https://github.com/GoogleCloudPlatform/knowledge-catalog/blob/main/okf/SPEC.md)). It records changes to the **knowledge** in `Atlas/` (concepts created, updated, removed). This is distinct from [[AI-OS/log|the vault work log]], which records operational sessions (ingests, lints, backups).

Newest dates first, ISO 8601. Prefix each entry with a conventional action label: `**Creation**`, `**Update**`, `**Removal**`.

## 2026-07-07

- **Creation** Atlas adopted the OKF v0.1 bundle changelog.
