# Calendar

The time layer: notes anchored to a date rather than a subject. The daily note is the single most important file in this vault, because it is where everything enters. The one habit this system asks of you: open today's daily note and dump everything there. The [[daily-review]] SOP sorts it later.

## Subfolders

| Folder | Holds |
|---|---|
| `Daily Notes/` | One note per day: log, captures, tasks |
| `Weekly Notes/` | One note per week: review and preview |
| `Meetings/` | One note per meeting, linked to people and projects |

## What belongs here

- Daily notes (`type: daily`), weekly notes (`type: weekly`), meeting notes (`type: meeting`).
- Anything time-bound: what happened, what was said, what you thought that day.

## What does not belong here

- Distilled knowledge (promote it to `Atlas/` via [[daily-review]] or [[ingest]]).
- Project plans (those live in `Efforts/` and get linked from daily notes).
