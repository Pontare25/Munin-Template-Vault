# System

Vault plumbing: files that make the vault work but are not knowledge themselves.

## Subfolders

| Folder | Holds |
|---|---|
| `Templates/` | Templater templates for every note type |
| `Attachments/` | Images and other media, kept out of the note folders |

## What belongs here

- Templates, attachments, Bases files shared across notes.

## What does not belong here

- Notes of any kind.
